# Module atomic_ghost Copy item path

<span class="sub-heading"><a href="../../src/vstd/atomic_ghost.rs.html#1-642"
class="src">Source</a> </span>

</div>

Expand description

<div class="docblock">

Provides sequentially-consistent atomic memory locations with associated
ghost state. See the
[`atomic_with_ghost!`](../macro.atomic_with_ghost.html "macro vstd::atomic_with_ghost")
documentation for more information.

</div>
